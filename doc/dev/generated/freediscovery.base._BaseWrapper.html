Base class for wrappers in FreeDiscovery

Parameters:
  • cache_dir (str) – folder where the model will be saved
  • parent_id (str, optional) – dataset id
  • mid (str, optional) – model id
  • dataset_definition (a dataset defintion class) – one of FeatureVectorizer, EmailParser
  • load_model (bool) – whether the model should be loaded from disk on class initialization
